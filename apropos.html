À propos du projet

Découvrez les détails techniques et l'objectif de ce projet.

Description du projet

Ce site web a été réalisé dans le cadre d'un projet scolaire de deuxième année en développement web. L'objectif est de créer une application simple permettant de gérer une liste d'étudiants sans utiliser de serveur ou de base de données externe.

Toutes les données sont stockées dans le localStorage du navigateur, ce qui permet de conserver les informations même après fermeture de la page. Le projet est entièrement développé avec les technologies de base du web : HTML, CSS et JavaScript.

Technologies utilisées

HTML5

Langage de base utilisé pour structurer le contenu des pages web. Chaque page est un fichier .html avec une structure sémantique claire.

CSS3

Utilisé pour le style et la mise en page. Le projet utilise Flexbox et CSS Grid pour créer une interface responsive et moderne.

JavaScript

Le JavaScript gère toute la logique : ajout et suppression d'étudiants, recherche, filtrage et sauvegarde dans le localStorage.

localStorage

Permet de sauvegarder les données directement dans le navigateur. Les étudiants restent disponibles même après fermeture et réouverture de la page.

✅ Fonctionnalités du projet

  • Affichage d'une liste dynamique d'étudiants dans un tableau
  • Ajout d'un étudiant via un formulaire avec validation
  • Suppression d'un étudiant avec confirmation
  • Recherche d'un étudiant par nom ou prénom
  • Filtrage de la liste par filière
  • Calcul automatique de la moyenne générale
  • Identification du meilleur étudiant
  • Répartition des étudiants par filière avec barres de progression
  • Sauvegarde des données avec localStorage
  • Design responsive adapté au mobile et au bureau
