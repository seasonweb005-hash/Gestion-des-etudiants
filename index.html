Présentation du projet

Ce site a été créé dans le cadre d'un projet scolaire. Il permet de gérer une liste d'étudiants avec leurs informations de base : nom, prénom, filière et note. Toutes les données sont sauvegardées directement dans votre navigateur grâce au localStorage.

  • Ajouter et supprimer des étudiants facilement
  • Rechercher un étudiant par nom ou prénom
  • Filtrer par filière
  • Consulter les statistiques de la classe
  • Données sauvegardées dans le navigateur
